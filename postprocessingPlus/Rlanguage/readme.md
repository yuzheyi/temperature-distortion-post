## 所有文件使用r语言编写

### 1. data.r

data.r处理AIP平均温度参数随时间的变化

输入文件需要report文件

运行修改需要指定文件路径

输出文件包括"输出结果.pdf"和"summary.txt",pdf文件为物理量随时间变化图，summary为统计平均值

### 2. matlabData.r

处理matlab计算结果，主要是将计算结果进行图像绘图，r语言画的好看

### 3. mfData.r

处理爆震管入口流量问题，统计进口边界条件的流量问题

### 4. pdc.r

爆震管出口参数画图

### 5.time.r

将时间转化为csv格式方便后续处理
